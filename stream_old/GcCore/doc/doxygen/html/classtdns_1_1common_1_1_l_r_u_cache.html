Detailed Description

template<typename Key, typename Value>
class tdns::common::LRUCache< Key, Value >

Least Recently Used cache. The last used pair is at the end of the cache and the least recently used is at the beginning of it. /!\ Be carefull when you loop over it. Almost all operations modify the container. /!\ This container uses the erase / copy to update a pair, prefere the use of pointer in the values.

Key of the pair. It MUST implement the operator == and MUST BE copyable. Value of the pair. It MUST BE copyable.
