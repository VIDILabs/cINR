Detailed Description

template<typename T, uint32_t Option = 0>
class tdns::common::DynamicArray3dHost< T, Option >

Class to create a static array on the CPU given cuda option.

Type of the data to store. Size on of the array on X axis. Size on of the array on Y axis. Size on of the array on Z axis. Option of the array. How the allocation is performed. The default value is a pinned memory allocation.
