Compute the callstack from the MRPD to the data cache for a given brick. It will say from the MRPD to which PT cache the brick is referenced. It means after the last PT cache, PT blocks will be required to reference the new brick. In the stack, if:

  • volumePosition == -2.f => nothing to add a this level.
  • volumePosition == -1.f => new block to add, nothing to remove.
  • volumePosition > -1.F => old block to remove, new block to add.
