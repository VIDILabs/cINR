Detailed Description

template<typename T>
class tdns::common::Singleton< T >

CRTP Based singleton implementation.

This class provides basic stuff to handle singleton. This implementation is not thread-safe.

Template Parameters
[T]The class / type to instantiate as a singleton. Nothing special required regarding its interface: default constructor if you intend to create the singleton without parameters or a valid variadic constructor if parameters are given.
