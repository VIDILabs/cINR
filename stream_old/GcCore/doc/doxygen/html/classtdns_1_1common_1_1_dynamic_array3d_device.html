Detailed Description

template<typename T>
class tdns::common::DynamicArray3dDevice< T >

Class to create a 3D array on the device memory.

Template Parameters
[T]Type of the data to store.

Create a 3D array on the device memory typed of T. Because it uses a thrust::device_vector<T>, it is possible to get the data stored on the device using accessor overload [] or ().

Example:

tdns::math::Vector3ui array_size(10, 20, 30);
tdns::common::DynamicArray3dDevice<uint32_t> array(array_size);
array[0] = 42; //same as array(tdns::math::Vector3ui(0, 0, 0)) or array(0, 0, 0)
