Detailed Description

template<typename T>
class tdns::gpucache::MultiResolutionPageDirectory< T >

Class for a Multi resolution page directory (MRPD). With t = 1 => 1 entry of the MRPD addresses 1 brick ! with t = 2 => 1 entry addresses 1 blocks of page table entry and 1 table entry addresses 1 brick.
