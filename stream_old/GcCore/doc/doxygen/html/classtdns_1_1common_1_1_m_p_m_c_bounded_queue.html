Detailed Description

template<typename T>
class tdns::common::MPMCBoundedQueue< T >

A lock free implementation of a multi producers, multi consumers queue.

This class implements a high-performance bounded concurrent queue that supports multiple producers, multiple consumers, and optional blocking. The queue has a fixed capacity which is fixed in the constructor.

Template Parameters
[T]The element type the MPMCBoundedQueue will apply.
Warning
[T] MUST be copyable.
See also
More information: http://www.1024cores.net/home/lock-free-algorithms/queues/bounded-mpmc-queue
