Detailed Description

template<typename T, int n, uint32_t ulp = 4>
class tdns::math::Vector< T, n, ulp >

Template class to create vector size of 2, 3, 4.

Template Parameters
[T]The type of the values in the vector.
[N]The size of the vector.
[ulp](Units in the Last Place) see: MathUtils.hpp

IT eases the use of a mathematics vector and provides basic vector operations. The default values of a vector are 0.

Example:

tdns::math::Vector3ui vec1(1), vec2(2); //Create vector of 3 * uint32_t
tnds::math::Vector3ui vec3 = vec1 + vec2;
(vec3 == tnds::math::Vector3ui(3, 3, 3)) // TRUE
